# Phishing-Analysis-Lab

For anyone who wants to practice phishing analysis, this lab contains five phishing email samples used to identify malicious indicators, spoofed domains, suspicious links, and common social engineering techniques.

For safer analysis:
- Use a virtual machine (VM)
- Keep remote content disabled
- Do not click suspicious links
- Do not open attachments on your host machine

If you do not trust the direct download links, you can manually download the samples from the official repository below:

## 📌 Requirements

To complete this phishing analysis lab, you will need:

- A code editor (recommended: Visual Studio Code)
- Basic understanding of email headers and structure (.eml format)
- Knowledge of common phishing indicators (spoofed domains, suspicious URLs, etc.)
- Ability to identify encoding techniques (URL encoding, Base64, etc.)
- Internet connection (for domain and URL reputation checks)
- Optional tools:
  - CyberChef (for decoding encoded strings)
  - WHOIS lookup tools (for domain investigation)
  - VirusTotal (for URL and attachment analysis)

## ⚠️ Safety Notice

- Do NOT open attachments directly on your host machine
- Use a safe environment (VM or sandbox) when analyzing emails
